# fukui_eatsafe

[ふくい安全・安心飲食店認証制度](https://fukui-anshin-ninsyou.com/certified.html)の認証店情報をマッピングするプロジェクトです。

## デモ
[ふくい安心飲食店マップ](https://code4fukui.github.io/fukui_eatsafe/)

## 機能
- 福井県内の認証店の所在地をマップ上に表示
- 認証店情報をCSVで提供

## 必要環境
Denoランタイムが必要です。

## 使い方
1. HTMLファイルをローカルサーバーで公開します。
2. 認証店情報はCSVファイルから確認できます。

## データ
[ふくい安全・安心飲食店認証制度 認証店一覧](https://fukui-anshin-ninsyou.com/certified.html)のデータを利用しています。
